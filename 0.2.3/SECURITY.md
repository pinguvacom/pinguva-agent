# Security Policy

Этот каталог публикуется для прозрачности и аудита кода агента Pinguva версии 0.2.3.

Если вы нашли уязвимость:
- не публикуйте её публично до исправления;
- отправьте описание, шаги воспроизведения и возможное влияние напрямую команде Pinguva по закрытому каналу, который используется для security-обращений;
- приложите версию агента и, если возможно, минимальный proof of concept.

Что важно:
- backend, SaaS-сервис и серверная логика Pinguva не входят в этот архив;
- этот каталог покрывает только код агента;
- токены, state-файлы и рабочие конфигурации не должны попадать в публичные issue или отчёты.
