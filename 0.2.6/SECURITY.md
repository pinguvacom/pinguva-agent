# Security Policy

## Русский

Этот архив публикуется для аудита исходного кода Pinguva Agent `0.2.6`.
Backend и SaaS-сервис Pinguva в него не входят.

Если вы нашли уязвимость, не публикуйте её до исправления. Передайте команде
Pinguva версию агента, минимальные шаги воспроизведения и оценку влияния по
закрытому каналу для security-обращений. Не прикладывайте токены, webhook URL,
state-файлы или рабочие конфигурации к публичным issue.

## English

This archive is published for security review of Pinguva Agent `0.2.6`. The
Pinguva backend and SaaS service are not included.

If you find a vulnerability, do not disclose it publicly before a fix is
available. Send the agent version, minimal reproduction steps and impact to the
Pinguva security contact through a private channel. Do not include tokens,
webhook URLs, state files or live configuration in public issues.
