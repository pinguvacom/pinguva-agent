# Security Policy

## Русский

Этот каталог публикуется для прозрачности и аудита агента `0.2.12`. Backend,
SaaS-сервис и внутренняя серверная логика Pinguva сюда не входят.

Если вы нашли уязвимость, не публикуйте её до исправления. Передайте команде
Pinguva описание, воспроизведение, возможное влияние и версию агента через
закрытый канал для security-обращений. Не прикладывайте токены, state-файлы,
рабочие конфигурации, webhook или содержимое `/root/.my.cnf`.

## English

This directory is published for transparency and security review of agent
`0.2.12`. It does not include the Pinguva backend, SaaS service or internal
server logic.

If you find a vulnerability, do not disclose it publicly before a fix is
available. Send the Pinguva team a description, reproduction, potential impact
and agent version through the private security reporting channel. Do not attach
tokens, state files, live configurations, webhooks or `/root/.my.cnf` contents.
